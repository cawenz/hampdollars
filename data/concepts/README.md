# Concept sketches

Drop any `*.geojson` files here. They'll auto-load as a toggleable "Concept sketches" overlay on the Land context and Developable land tabs at the next app start.

How to create one:
1. On the Land context tab, use the draw toolbar (top-left of the map) to sketch polygons, rectangles, or lines.
2. Click "Download drawn shapes" in the sidebar to save the current drawings as a GeoJSON file.
3. Rename the file something meaningful (e.g., `2026-housing-vision.geojson`) and drop it in this folder.
4. Restart the app. The overlay picks it up automatically; the filename becomes the layer label.

Concepts drawn directly in the app are already georeferenced. If someone hands you a printed sketch, the easiest re-import is to retrace it using the app's draw tool with the scan as a visual reference.
